## Il classico Hello World

Iniziamo sempre dalle basi:

```c
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
```

Un semplice programma in C che stampa il famoso "Hello, world!" - il punto di partenza di ogni programmatore che si rispetti.
